Welcome to AI Documenter

This wizard will set up your AI documentation system. You'll need:

• Your Canvas API token
• A Gemini API key

The setup takes about 2 minutes.

Canvas Connection

Connect to your Canvas LMS to pull course and assignment data.

Generate at: Canvas → Account → Settings → New Access Token

Select Courses

Choose which courses to track AI usage for.

Courses will appear after connecting to Canvas.

Gemini API Setup

Connect to Google's Gemini AI for transcript processing.

Get one free at: aistudio.google.com → Get API Key

AI Tools List

Customize the dropdown list of AI tools students can select. Separate with commas. "Other" will always include a fill-in field.

Finishing Setup...

Creating course tabs, syncing assignments, and setting up triggers...
